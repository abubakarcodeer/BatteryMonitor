# 📱 Battery Monitor App (Kotlin)

## 📌 Overview
The **Battery Monitor App** is an Android application developed using **Kotlin** that provides real-time information about the device's battery status. It allows users to monitor battery percentage, temperature, charging status, and power source.

## 🚀 Features

- 🔋 **Battery Percentage**
  - Displays the current battery level

- 🌡️ **Battery Temperature**
  - Shows battery temperature in °C

- ⚡ **Charging Status**
  - Charging
  - Discharging
  - Fully Charged

- 🔌 **Charging Source**
  - USB
  - AC Charger
  - Wireless

- 🔄 **Real-Time Updates**
  - Automatically updates using system broadcasts

## 🛠️ Technologies Used

- **Language:** Kotlin
- **Platform:** Android
- **Components:**
  - BroadcastReceiver
  - IntentFilter (`ACTION_BATTERY_CHANGED`)
- **UI:** XML Layouts
